# check-in

## 小程序介绍
  + 点名时将位置信息和周边无线AP的SSID、信号强度上报
  + 服务器端设置上报的无线AP的信息数目 设置自动上报信息的频率
  + 客户端可以设置小程序自动启动的时间 （可以根据课表进行设置）
